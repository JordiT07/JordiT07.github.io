FreeCodeCamp JavaScript Curriculum

Section Steps Description
Code Editors 5 steps Introduces working with code editors and IDEs, such as VS Code, and how to configure your environment for efficient JavaScript development.
Variables and Strings 62 steps Explains JavaScript variables, strings, and data types through theory, reviews, and hands-on projects like chatbots and trivia games.
Booleans and Numbers 37 steps Covers arithmetic, comparison operators, boolean logic, conditionals, and math methods, reinforced with review quizzes and coding labs.
Functions 38 steps Introduces function syntax, parameters, return values, and logic organization through practical projects like calculators and converters.
Arrays 35 steps Focuses on array creation and manipulation using methods like push(), map(), and filter() with multiple labs and reviews.
Objects 28 steps Teaches JavaScript object structure, JSON basics, destructuring, and chaining, applied through recipe and quiz-building exercises.
Loops 22 steps Demonstrates for, while, and for...of loops to iterate data, build algorithms, and solve pattern-based coding challenges.
JavaScript Fundamentals Review 30 steps Consolidates earlier lessons by combining arrays, functions, and conditionals into algorithm labs and small project applications.
Higher Order Functions and Callbacks 37 steps Introduces callbacks and higher-order functions for advanced data manipulation, functional programming, and algorithm design.
DOM Manipulation and Events 87 steps Explores how to dynamically update webpages, respond to user input, and build interactive apps like a storytelling game and emoji reactor.
JavaScript and Accessibility 42 steps Focuses on building inclusive interfaces using ARIA attributes and accessibility features with projects like a tablist and theme switcher.
Debugging 8 steps Introduces debugging tools and strategies to identify and fix JavaScript errors using console methods and browser dev tools.
Basic Regex 47 steps Teaches regular expressions for pattern matching, validation, and string manipulation with creative labs like a spam filter and palindrome checker.
Form Validation 103 steps Explains client-side validation and user input handling through projects like a calorie counter and customer complaint form.
Dates 5 steps Covers working with JavaScript’s Date object and time conversion with a small date formatting lab and review quiz.
Audio and Video Events 55 steps Demonstrates controlling media elements using event listeners, creating a music player, and building a simple drum machine app.
Maps and Sets 37 steps Introduces ES6 data structures for key-value storage and unique collections through projects like a voting system and plant catalog.
LocalStorage and CRUD Operations 78 steps Explains client-side persistence using localStorage, implementing Create, Read, Update, Delete (CRUD) with projects like a to-do app and bookmark manager.
Classes 68 steps Covers ES6 classes, constructors, and methods through structured exercises building apps such as a shopping cart and bank manager.
Recursion 112 steps Breaks down recursive function design, call stack logic, and problem-solving with projects like binary converters and permutation generators.
Functional Programming 23 steps Emphasizes immutability, pure functions, and chaining techniques applied to data transformation and algorithm optimization.
Asynchronous JavaScript 36 steps Teaches asynchronous operations with callbacks, Promises, and async/await through projects like weather apps and leaderboards.
JavaScript Review 1 step Comprehensive review covering all major concepts — from variables and loops to DOM manipulation — before the prep exam.
